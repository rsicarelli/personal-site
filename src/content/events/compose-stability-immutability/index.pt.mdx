O Compose é um framework de UI declarativa que possibilita criar telas para múltiplas plataformas,
incluindo Android, iOS, Desktop e Web/Wasm. Mas você sabia que o Compose é, na verdade, um gerador
de estado com superpoderes — e que o Compose UI usa esse gerador para construir os componentes
visuais? Nesta palestra aprofundo os conceitos de Estabilidade e Imutabilidade no Compose Runtime:
como impactam o desempenho, como a estabilidade permite ao Compose evitar recomposições
desnecessárias e tornar as telas mais fluidas.
