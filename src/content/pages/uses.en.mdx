A living list of the tools I rely on for development, writing and talks. It changes over time —
this is a snapshot.

## Editor & terminal

- **Editor:** IntelliJ IDEA for Kotlin/KMP; VS Code for web and writing.
- **Terminal:** a Nerd Font with a minimal prompt, plus `mise` for toolchain pinning.

## Languages & platforms

Kotlin and Kotlin Multiplatform first, with Swift, TypeScript, Python and Rust as the work
demands. Gradle and GitHub Actions hold the builds together.

## Hardware

A MacBook for daily work and an external display for long writing and review sessions.
