Uma lista viva das ferramentas em que confio para desenvolvimento, escrita e palestras. Ela muda
com o tempo — este é um retrato do momento.

## Editor e terminal

- **Editor:** IntelliJ IDEA para Kotlin/KMP; VS Code para web e escrita.
- **Terminal:** uma Nerd Font com prompt minimalista, além do `mise` para fixar o toolchain.

## Linguagens e plataformas

Kotlin e Kotlin Multiplatform em primeiro lugar, com Swift, TypeScript, Python e Rust conforme o
trabalho pede. Gradle e GitHub Actions seguram os builds.

## Hardware

Um MacBook para o trabalho diário e um monitor externo para longas sessões de escrita e revisão.
